# Scroll Capture (C)

## Скачать

```
c/dist/scroll_capture_v1.0.23.exe
```

## Kaspersky / антивирус

Утилита делает скриншоты и прокрутку мыши для VRM/RDP — без цифровой подписи AV может ругаться. Добавьте exe в **исключения** или соберите сами: `cd c && build.bat`

## Прокрутка

Адаптивный скролл: **22–32%** нового контента за кадр, 1 щелчок колёсика за micro-step (одинаково для всех экранов).

## Выходной файл

`long_screenshot_YYYY-MM-DD_HH-MM-SS.png`
