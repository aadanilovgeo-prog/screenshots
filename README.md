# Scroll Capture (C)

Утилита для Windows (VRM / RDP / виртуальная машина): прокрутка колёсиком, серия скриншотов, **Safe Stitch** — склейка с приоритетом сохранения контента.

## Скачать

```
c/dist/scroll_capture_v1.0.2.exe
```

## Быстрый старт

1. Прокрутите статью в начало.
2. Запустите exe.
3. Выберите область (левый верхний → правый нижний угол).
4. Переключитесь в окно VRM после отсчёта.

Итоговый файл: `long_screenshot_YYYY-MM-DD_HH-MM-SS.png`

## Safe Stitch (по умолчанию)

- Смещение контента определяется **по кадрам**, а не по `wheel_notches × 35px`.
- При сомнении crop **уменьшается** — лучше небольшой дубль, чем пропуск строк таблицы или части изображения.
- Каждый шов оценивается **отдельно**, без медианного сглаживания по всем кадрам.
- `wheel_notches` — только физическая прокрутка мыши.

## Отладка

```bat
scroll_capture_v1.0.2.exe --save-frames ./debug
```

Сохраняет: `frame_NNNN.png`, `seam_NNNN_preview.png`, `stitch_log.txt`.

## Сборка

```bat
cd c
build.bat
```

Версия повышается только при мерже в `main` (CI). Локальный `build.bat` версию не меняет.

## Параметры

| Параметр | Описание |
|----------|----------|
| `--wheel-notches 10` | Щелчков колёсика (прокрутка) |
| `--scroll-delay 0.8` | Пауза после прокрутки |
| `--save-frames DIR` | Кадры + preview швов + лог |
| `--no-safe-stitch` | Отключить безопасный режим (не рекомендуется) |
| `--region L,T,W,H` | Область захвата |
| `-o file.png` | Выходной файл |

Полный список: `scroll_capture_v1.0.2.exe --help`
