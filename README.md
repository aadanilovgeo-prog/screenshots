# Scroll Capture (C)

## Скачать

```
c/dist/scroll_capture_v1.0.10.exe
```

## Прокрутка (физическая)

| Профиль | Высота области | Щелчков за шаг | Цель (новый контент) |
|---------|----------------|----------------|----------------------|
| Большой экран | > 1000 px | **2** | 22–32% |
| Малый (~13″) | ≤ 1000 px | **4** | 44–64% |

## ESC — частичная склейка

Во время захвата нажмите **ESC** — программа **остановит прокрутку** и **склеит уже собранные кадры** в PNG (не выходит без результата).

## Safe Stitch

Склейка по смещению контента; при сомнении — меньше crop.

## Отладка

```bat
scroll_capture_v1.0.10.exe --save-frames ./debug
```

## Выходной файл

`long_screenshot_YYYY-MM-DD_HH-MM-SS.png`
