# Scroll Capture (C)

Утилита для Windows (VRM / RDP): **адаптивная прокрутка** + **Safe Stitch**.

## Скачать

```
c/dist/scroll_capture_v1.0.3.exe
```

## Как работает прокрутка

- **1 щелчок колёсика** за раз, пока новый контент не составит **22–32%** высоты области захвата.
- Не зависит от дюймов монитора — только от **высоты выделенной области**.
- Параметр `wheel_notches` **убран**.

## Safe Stitch

- Склейка по фактическому смещению контента между кадрами.
- При сомнении crop уменьшается — лучше дубль, чем пропуск строки таблицы.

## Отладка

```bat
scroll_capture_v1.0.3.exe --save-frames ./debug
```

## Выходной файл

`long_screenshot_YYYY-MM-DD_HH-MM-SS.png`

## Сборка

```bat
cd c
build.bat
```
