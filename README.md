# Scroll Capture (C)

## Скачать

```
c/dist/scroll_capture_v1.0.17.exe
```

## Kaspersky / антивирус

Утилита делает скриншоты и прокрутку мыши для VRM/RDP — без цифровой подписи AV может ругаться. Добавьте exe в **исключения** или соберите сами: `cd c && build.bat`

## Прокрутка

| Профиль | Высота области | Щелчков за шаг | Цель |
|---------|----------------|----------------|------|
| Большой экран | > 1000 px | 3 | 22–32% |
| Малый (~13″) | ≤ 1000 px | 3 | 44–64% |

## Выходной файл

`long_screenshot_YYYY-MM-DD_HH-MM-SS.png`
