# Scroll Capture (C)

Утилита для Windows (VRM / RDP): **адаптивная прокрутка** + **Safe Stitch**.

## Скачать

```
c/dist/scroll_capture_v1.0.8.exe
```

## Адаптивная прокрутка

| Профиль | Высота области | Новый контент за кадр |
|---------|----------------|------------------------|
| Большой экран (~34″) | > 1000 px | **44–64%** |
| Малый экран (~13″) | ≤ 1000 px | **66–82%** |

1 щелчок колёсика за раз, пока сдвиг не попадёт в диапазон.

## Safe Stitch

Склейка по фактическому смещению контента; при сомнении — меньше crop, лучше дубль чем пропуск.

## Отладка

```bat
scroll_capture_v1.0.8.exe --save-frames ./debug
```

## Выходной файл

`long_screenshot_YYYY-MM-DD_HH-MM-SS.png`
