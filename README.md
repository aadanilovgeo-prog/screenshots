# Scroll Capture (C)

## Скачать

```
c/dist/scroll_capture_v1.0.12.exe
```

## Kaspersky / антивирус

Утилита **не содержит вредоносного кода**. Kaspersky и другие AV могут ругаться, потому что программа:

- делает скриншоты области экрана;
- эмулирует прокрутку мыши (для VRM/RDP);
- распространяется **без цифровой подписи** (unsigned exe).

**v1.0.12+:** сборка через MSVC с VERSIONINFO и manifest, `SendInput` вместо устаревшего `mouse_event`.

### Что делать

1. Добавьте exe в **Исключения** Kaspersky (Угрозы и исключения → Исключения → Добавить файл).
2. Или сообщите о **ложном срабатывании**: [Kaspersky — False Positive](https://opentip.kaspersky.com/)

Исходный код открыт в этом репозитории — можно собрать самостоятельно: `cd c && build.bat`

## Прокрутка

| Профиль | Высота области | Щелчков за шаг | Цель |
|---------|----------------|----------------|------|
| Большой экран | > 1000 px | 2 | 22–32% |
| Малый (~13″) | ≤ 1000 px | 4 | 44–64% |
