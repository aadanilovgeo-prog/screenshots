# scroll_capture (C)

## Download

```
c/dist/scroll_capture_v1.0.8.exe
```

## Adaptive scroll

| Profile | Capture height | New content per frame |
|---------|----------------|------------------------|
| Large screen (~34") | > 1000 px | 44–64% |
| Small screen (~13") | ≤ 1000 px | 66–82% |

## Debug

```bat
scroll_capture_v1.0.8.exe --save-frames ./debug
```
