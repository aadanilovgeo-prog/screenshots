# scroll_capture (C)

## Download

```
c/dist/scroll_capture_v1.0.6.exe
```

## Adaptive scroll

| Profile | Capture height | New content per frame |
|---------|----------------|------------------------|
| Default (large screen) | > 1000 px | 22–32% |
| Small screen (~13") | ≤ 1000 px | 44–64% (2x step) |

## Debug

```bat
scroll_capture_v1.0.6.exe --save-frames ./debug
```
